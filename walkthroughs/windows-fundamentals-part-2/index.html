Task 2: System Configuration

Explore tools for configuring and troubleshooting your Windows system.

1. What is the name of the service that lists Systems Internals as the manufacturer?
Ans: PsShutdown

2. Whom is the Windows license registered to?
Ans: Windows User

3. What is the command for Windows Troubleshooting?
Ans: C:\\Windows\\System32\\control.exe /name Microsoft.Troubleshooting

4. What command will open the Control Panel?
Ans: control.exe

Task 3: Change UAC Settings

Learn how to control when you're notified about changes to your computer.

1. What is the command to open User Account Control Settings?
Ans: UserAccountControlSettings.exe

Task 4: Computer Management

Use the Computer Management console to inspect scheduled tasks and shared folders.

1. What is the command to open Computer Management?
Ans: compmgmt.msc

2. At what time every day is the GoogleUpdateTaskMachineUA task configured to run?
Ans: 6:15 AM

3. What is the name of the hidden folder that is shared?
Ans: sh4r3dF0Ld3r

Task 5: System Information

The System Information tool gives a detailed overview of your system’s configuration.

1. What is the command to open System Information?
Ans: msinfo32.exe

2. What is listed under System Name?
Ans: THM-WINFUN2

3. Under Environment Variables, what is the value for ComSpec?
Ans: %SystemRoot%\system32\cmd.exe

Task 6: Resource Monitor

Monitor how your system resources (CPU, memory, disk, and network) are used.

1. What is the command to open Resource Monitor?
Ans: resmon.exe

Task 7: Command Prompt

Dive deeper into networking commands using the command line interface.

1. In System Configuration, what is the full command for Internet Protocol Configuration?
Ans: C:\\Windows\\System32\\cmd.exe /k %windir%\\system32\\ipconfig.exe

2. For the ipconfig command, how do you show detailed information?
Ans: ipconfig /all

Task 8: Registry Editor

1. What is the command to open the Registry Editor?
