Task 2: Windows Editions

This task introduces the different editions of Windows OS and features that distinguish them.

1. What encryption can you enable on Pro that you can't enable in Home?
Ans: BitLocker — A full-disk encryption feature only available on Pro and Enterprise editions of Windows.

Task 3: The Desktop (GUI)

Explore the graphical user interface of Windows including the Start Menu, Taskbar, and Notification Area.

1. Which Selection will hide/disable the Search box?
Ans: Hidden

2. Which selection will hide/disable the task view button?
Ans: Show Task View Button — Toggling this hides/shows the multitasking view icon.

3. Besides Clock and Network, what other icon is visible in the Notification Area?
Ans: Action Center — This holds notifications and quick access toggles.

Task 4: The File System

Understand how files are stored, accessed, and managed in Windows.

1. What is the meaning of NTFS?
Ans: New Technology File System — A modern file system offering security, journaling, and support for large volumes.

Task 5: The Windows\System32 Folders

Learn about a crucial system directory that holds OS and system tool executables.

1. What is the system variable for the windows folder?
Ans: %windir% — This environment variable points to the Windows installation directory.
